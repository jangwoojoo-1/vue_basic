1. 컴포지션 API
  1-1. 데이터 정의 방법
    - 반응형 데이터 정의 : ref(), reactive()
      ref() : 기본 자료형, 참조 자료형 구분없이 사용 가능하지만 .value 속성으로 접근하기 때문에 참조형 타입에 반응성이 전달되지 않는다.
      reactive() : 참조 자료형에 반응성을 부여할 때 사용한다. 

  1-2. 템플릿 ref
    - ref() 함수로 생성한 반응형 데이터를 HTML 태그에서 접근할 때 사용한다.
      ref 속성의 값으로 지정하여 DPM 요소를 참조할 수 있다.
